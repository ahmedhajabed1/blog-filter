== Description ==

**AJAX Post Filter** is a powerful WordPress plugin that allows you to filter blog posts by categories without page reloads. Perfect for blogs, news sites, portfolios, and any content-heavy website.

= Key Features =

* **AJAX Filtering** - Filter posts without page reload for seamless user experience
* **Elementor Compatible** - Drag and drop widget with live preview
* **Lazy Loading** - Improve page speed by 50-70% with automatic image lazy loading
* **3 Pagination Types** - Standard pagination, Load More button, or Infinite Scroll
* **SEO Optimized** - Automatic rel="prev" and rel="next" meta tags
* **Fully Customizable** - Change button text, colors, and styles
* **Multiple Presets** - Create unlimited filter configurations
* **Responsive Design** - Works perfectly on all devices
* **Category Search** - Built-in search functionality
* **Active Filters Display** - Show which filters are applied
* **Professional Admin Interface** - Beautiful YITH-style admin panel

== Installation ==

= Automatic Installation =

1. Go to Plugins → Add New
2. Search for "AJAX Post Filter"
3. Click "Install Now"
4. Activate the plugin

= Manual Installation =

1. Download the plugin ZIP file
2. Go to Plugins → Add New → Upload Plugin
3. Choose the ZIP file and click "Install Now"
4. Activate the plugin

= After Activation =

1. Go to **Post Filter** in WordPress admin
2. Configure your first preset
3. Copy the shortcode OR use Elementor widget
4. Add to any page and enjoy!

== Frequently Asked Questions ==

= Does this work with Elementor? =

Yes! The plugin includes a dedicated Elementor widget with full visual controls and live preview.

= Can I use this without Elementor? =

Absolutely! You can use shortcodes on any page, post, or widget area.

= How do I add the filter to my page? =

**Method 1 (Elementor):** Drag the "Ajax Post Filter" widget to your page

**Method 2 (Shortcode):** Use `[ajax_post_filter slug="default-preset"]`

= Can I translate the button text? =

Yes! All button text is customizable in the admin panel under Customization → Button Text.

= Does it support lazy loading? =

Yes! Lazy loading is built-in and can be toggled per preset or per widget.

= What pagination types are supported? =

Three types: Standard Pagination, Load More Button, and Infinite Scroll.

= Is it SEO-friendly? =

Yes! The plugin automatically adds proper rel="prev" and rel="next" meta tags for pagination.

= Can I customize the colors? =

Yes! You can customize primary color, button colors, text colors, and hover colors.

= How many presets can I create? =

Unlimited! Create as many filter presets as you need.

= Does it work on mobile? =

Yes! Fully responsive and touch-optimized for all devices.

= Can I filter custom post types? =

The current version filters standard WordPress posts. Custom post type support coming in future updates.

= Is it compatible with caching plugins? =

Yes! The AJAX functionality works with most caching plugins.
